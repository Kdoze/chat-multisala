# Chat com Salas (Trabalho de Socket)

Este projeto foi feito para um trabalho da disciplina, com o objetivo de criar um sistema de **chat com múltiplas salas** usando **Node.js** e **Socket.io**.

## O que foi feito

- Uma página HTML com um campo para nickname.
- Um combobox com 4 salas diferentes.
- Um botão para o usuário entrar na sala.
- Quando o usuário entra, ele pode enviar mensagens.
- As mensagens são recebidas **somente** pelos usuários que estão na **mesma sala**.
- O back-end foi feito com **Express** e **Socket.io**.
